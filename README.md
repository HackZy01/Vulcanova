# Vulcanova
Nieoficjalny, międzyplatformowy, mobilny klient dziennika Vulcan UONET+ tworzony w Xamarinie. Aktualnie w fazie intensywnego rozwoju.

## Co działa:
- szczęśliwy numerek
- oceny
- plan lekcji
- obecność/frekwencja

## Co jest planowane:
- wiadomości
- dashboard
- prace domowe/zadania
- sprawdziany/kartkówki
- punkty (za zachowanie)
I wiele więcej!
